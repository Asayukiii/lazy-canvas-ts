## Example

```js
const { LazyCanvas, NgonLayer } = require('@hitomihiumi/lazy-canvas')

let ngon = new NgonLayer()
.setRadius(50)
.setSides(6)
.setColor('#fff')

const lazy = new LazyCanvas()
.addLayers(ngon)
//...
```
