Extends [BaseLayer](./baselayer.md)

## Example

```js
const { LazyCanvas, RectangleLayer } = require('@hitomihiumi/lazy-canvas')

let rectangle = new RectangleLayer()
.setX(10)
.setY(10)
.setWidth(200)
.setHeight(100)
.setColor('#FF0000')

const lazy = new LazyCanvas()
.addLayers(rectangle)
//...
```
