## Example

```js
const { LazyCanvas, CircleLayer } = require('@hitomihiumi/lazy-canvas')

let circle = new CircleLayer()
.setX(100)
.setY(100)
.setRadius(100)
.setColor('red')

const lazy = new LazyCanvas()
.addLayers(circle)
//...
```
